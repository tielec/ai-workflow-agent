## Output Requirements

Generate a commit message in this format:

```
<type>(<scope>): <subject>

<body>

Fixes #{issue_number}
```

### Details
- **`<type>`**: one of
  - `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`
- **`<scope>`** (optional): affected area (e.g., `git`, `metadata`, `evaluation`)
- **`<subject>`**:
  - <= 50 characters, imperative mood, lowercase start, no period
- **`<body>`** (optional): why this change is needed and what it fixes
- **`Fixes #{issue_number}`**: required issue reference

## Output Instruction

Write the generated commit message to `.ai-workflow/tmp/squash/commit-message.txt` using the write tool.
