{

      "patternName": "パターン名",
      "description": "発見事項の説明",
      "severity": "info|warning",
      "evidence": [
        {
          "type": "code_reference|file_content|git_history|search_result",

}

- 証拠がない場合は findings を空配列にしてください。

- 推測ではなく事実ベースで記載してください。
