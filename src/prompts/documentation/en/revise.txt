## Correction policy

### 1. Thorough document search

Please double check the following locations:

- .md file directly under the project root
- .md file in a subdirectory (excluding the .ai-workflow directory)
- docs directory (if available)

### 2. Review your update decision

For each document, answer the following questions again:

1. **Do readers of this document need to know about this change? **
2. **Will readers be in trouble if they don't know? Do you misunderstand? **
3. **Is the content of the document outdated? **

→ **If yes to any of the above, update target**

### 3. Improved update content

While maintaining the existing style and format:

- Describe the changes concisely
- Describe accurate information (consistent with Phases 2 and 4)
- Focus on useful information from the user's perspective

### 4. Improved update log

Documentation-update-log.md contains the following:
